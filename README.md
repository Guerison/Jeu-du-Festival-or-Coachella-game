# 🎤 Jeu du Festival

Un jeu stratégique d’enchères inspiré du poker où les joueurs doivent construire le meilleur festival avec un budget limité.

---

## 🚀 Concept

Chaque joueur dispose d’un budget (50M€) pour recruter des artistes.

Le jeu combine :

* 💰 **Enchères classiques**
* 🧠 **Stratégie et bluff (inspiré du poker)**
* 🎯 **Gestion de budget**

---

## 🎮 Règles du jeu

### 1. Objectif

Créer le meilleur line-up d’artistes avec un budget limité.

---

### 2. Déroulement

Pour chaque artiste :

* Une mise de départ est fixée
* Les joueurs jouent chacun leur tour

---

### 3. Actions possibles

#### 🔹 Phase normale :

* **Enchérir** → augmenter le prix
* **Rester** → rester dans le tour sans enchérir
* **Laisser** → quitter → abandonner l’artiste

#### 🔹 Phase poker (après une enchère) :

* **Suivre (Call)** → rester dans le jeu sans augmenter
* **Relancer (Raise)** → augmenter la mise
* **Passer (Fold)** → abandonner

---

### 4. Fin d’un tour

Quand tous les joueurs ont répondu à la dernière enchère :

* Le dernier joueur ayant relancé remporte l’artiste
* Il paie le prix final

---

### 5. Fin de partie

Quand tous les artistes sont attribués :

* Chaque joueur a son festival
* Le gagnant est choisi selon les préférences (vote ou système de points)

---

## 🖥️ Fonctionnalités

* Interface graphique avec Tkinter
* Gestion de plusieurs joueurs
* Système d’enchères dynamique
* Mode poker intégré
* Tableau des scores en temps réel
* Attribution automatique des artistes
